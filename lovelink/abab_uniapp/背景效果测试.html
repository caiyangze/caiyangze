🌟 现代化背景效果测试

这是用户详情页面的现代化背景效果展示，包含星光闪烁动画和渐变流动效果。

✨ 背景特效包含：

  • 🎨 深色渐变背景 - 紫色系主题色彩
  • ⭐ 星光闪烁动画 - 动态星光效果
  • 🌊 渐变流动效果 - 彩色光晕动画
  • 💫 毛玻璃效果 - 现代化视觉层次
  • 🎯 响应式设计 - 适配各种设备

🎯 设计理念：

采用现代化的设计语言，通过多层背景叠加、动态效果和毛玻璃材质，营造出科技感十足的视觉体验。色彩搭配以紫色为主调，辅以蓝色和粉色，形成和谐统一的视觉效果。

🚀 技术实现：

  • 使用 CSS3 linear-gradient 创建渐变背景
  • 使用 radial-gradient 实现星光点缀效果
  • 使用 @keyframes 创建流畅的动画效果
  • 使用 backdrop-filter 实现毛玻璃效果
  • 使用 z-index 管理层级关系
